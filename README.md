■ツールの概要

【アプリタイトル】remAInd

【ジャンル】学習用ツール

【特徴】キーワードと難易度を設定して四択問題を自動生成し、AIが回答を自動判定・解説します。生成した問題は履歴として保存され、解説の再閲覧や再挑戦、初回解答時の正誤確認が可能な学習支援ツールです。
